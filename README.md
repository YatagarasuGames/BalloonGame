# BalloonGame
У нас есть воздушный шарик, который поднимается в небе, встречая на пути падающие на него шипы, если шар касается шипов - игра заканчивается. Со временем скорость подъема шара увеличивается, уворачиваться становится сложнее.
Есть бонусы, которые дают различные плюшки(x2 бонус очков, неуязвимость, доп очки и тд)
Цель игры - подняться как можно выше чтобы:
1) Подняться выше в таблице лидеров(заносится только лучший счет)
2) Получать очки чтобы купить всю косметику в игре(условно, доп скины на шар)

Релиз будет на яндекс игры с интеграцией их лидербордов и системы рекламы/покупок.
